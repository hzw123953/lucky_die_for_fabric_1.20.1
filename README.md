# 幸运死亡模组 (Lucky Die Mod)

一个有趣的 Minecraft Fabric 模组，为游戏添加了随机死亡机制！

## 功能特性

### 服务器端功能
- **万分之一几率死亡**：服务器上每十秒有万分之一的概率随机选择一个在线玩家执行死亡
- **死亡不掉落**：死亡的玩家不会掉落任何物品，包括背包物品和末影箱物品
- **经验清零**：死亡后玩家的经验值会被清零
- **全服广播**：当有玩家被"幸运选中"时，会向全服务器广播消息

### 客户端功能
- **进入存档后启动**：玩家进入存档后，每十秒有万分之一的概率执行死亡
- **死亡不掉落**：客户端死亡同样不会掉落任何物品
- **经验清零**：死亡后经验值会被清零
- **个人通知**：被选中的玩家会在聊天栏收到通知消息

## 安装说明

### 前置要求
- Minecraft 1.20.1
- Fabric Loader
- Fabric API

### 安装步骤
1. 确保已安装 Fabric Loader 和 Fabric API
2. 下载本模组的 `.jar` 文件
3. 将模组文件放入服务器的 `mods` 文件夹中
4. 重启服务器

## 使用方法

- 服务器启动后，模组会自动开始工作
- 玩家正常进入游戏即可触发随机死亡机制
- 无需额外配置，模组会自动运行

## 注意事项

⚠️ **重要提醒**：
- 死亡的玩家会丢失所有背包物品和经验值
- 死亡机制对创造模式玩家同样有效
- 建议在测试服或娱乐服使用，避免在重要存档中使用
- 万分之一几率意味着平均每10000秒（大约2.8小时）会有一次死亡发生

## 配置文件

目前模组没有配置文件，所有参数都是硬编码的：
- 死亡间隔：10秒（200 ticks）
- 死亡几率：0.01%（万分之一）
- 死亡方式：世界边界伤害
